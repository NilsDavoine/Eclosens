# Create FAQ Schema Workflow

## Objective
Generate SEO FAQ sections and JSON-LD schema markup.

## Inputs
- page topic
- user concerns
- target keywords

## Instructions
1. Generate relevant FAQ questions
2. Write reassuring answers
3. Keep answers concise
4. Add FAQ JSON-LD schema
5. Ensure SEO relevance

## Output
- FAQ HTML section
- JSON-LD schema

## Edge Cases
- Duplicate FAQs
- Medical claims
- Overly technical language
