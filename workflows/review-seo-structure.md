# Review SEO Structure Workflow

## Objective
Review the SEO quality and architecture of the Éclosens website.

## Inputs
- sitemap
- page structure
- internal links
- metadata

## Instructions
1. Review heading hierarchy
2. Verify internal linking
3. Check keyword targeting
4. Detect duplicate intent
5. Verify metadata quality
6. Review semantic structure
7. Ensure local Luxembourg SEO presence

## Output
- SEO review report
- detected issues
- optimization recommendations

## Edge Cases
- Cannibalized keywords
- Thin content
- Missing metadata
- Broken internal links
