# Optimize Meta Tags Workflow

## Objective
Generate optimized SEO metadata for a page.

## Inputs
- page title
- target keyword
- page description

## Instructions
1. Create SEO optimized title tag
2. Write compelling meta description
3. Add canonical URL
4. Generate Open Graph tags
5. Ensure keyword relevance
6. Keep titles readable and human

## Output
- title tag
- meta description
- Open Graph tags
- canonical tag

## Edge Cases
- Duplicate titles
- Keyword stuffing
- Overly long metadata
