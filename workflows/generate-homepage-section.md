# Generate Homepage Section Workflow

## Objective
Create emotionally supportive homepage sections for Éclosens.

## Inputs
- section purpose
- emotional goal
- CTA objective

## Instructions
1. Use calm and premium tone
2. Prioritize emotional connection
3. Maintain airy readability
4. Include clear CTA
5. Keep paragraphs concise
6. Match brand identity

## Output
- HTML section
- heading
- supporting copy
- CTA content

## Edge Cases
- Overly dense text
- Aggressive marketing tone
- Repetitive emotional messaging
