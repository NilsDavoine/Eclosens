# Generate SEO Page Workflow

## Objective
Create a long-form SEO optimized HTML page for the Éclosens website.

## Inputs
- target keyword
- page title
- page purpose
- emotional objective
- internal links

## Instructions
1. Create semantic HTML structure
2. Add SEO title and meta description
3. Use proper heading hierarchy
4. Include emotionally supportive copywriting
5. Include FAQ section
6. Add internal linking
7. Include JSON-LD schema if relevant
8. Optimize for readability and spacing

## Output
- complete HTML page
- SEO metadata
- FAQ schema
- optimized semantic structure

## Edge Cases
- Missing target keyword
- Overlapping existing page topic
- Duplicate SEO intent
