# Eclosens — Claude Rules

# Brand Identity

The website represents Eclosens:
a premium emotional accompaniment brand dedicated to IVF/FIV and PMA support.

The website must feel:
- calm
- reassuring
- emotionally intelligent
- soft
- premium
- human
- elegant
- trustworthy

Never:
- aggressive
- cold
- corporate
- overly clinical
- spiritual
- influencer-like

---

# Emotional Direction

The visitor should feel:
- emotionally safe
- understood
- reassured
- calmer
- supported

The website should emotionally feel like:
- a calm refuge
- a cocoon
- a breathing space

---

# Writing Rules

Always:
- use emotionally intelligent language
- keep paragraphs breathable
- maintain calm pacing
- prioritize readability
- write naturally

Never:
- keyword stuff
- use aggressive marketing
- use spiritual vocabulary
- use fake urgency
- overpromise
- use miracle language

Forbidden vocabulary:
- manifestation
- healing energy
- abundance
- vibrations
- feminine energy
- miracle fertility
- unlock fertility

---

# SEO Rules

Always:
- use semantic HTML
- include meta descriptions
- maintain heading hierarchy
- use internal linking
- optimize naturally
- include Luxembourg naturally when relevant

Never:
- overoptimize
- force keywords unnaturally
- create thin content

---

# Design Rules

The UI must feel:
- airy
- calm
- warm
- elegant
- breathable

Always:
- use generous whitespace
- maintain soft visual hierarchy
- prioritize readability
- use subtle animations only

Avoid:
- clutter
- harsh black
- flashy effects
- overdecorated layouts
- heavy gradients
- hard shadows

---

# Photography Rules

Use imagery that feels:
- authentic
- calm
- intimate
- emotionally warm
- natural

Avoid:
- fake medical stock photos
- yoga/spa clichés
- overly clinical visuals
- overly happy fake imagery

---

# Technical Rules

Prioritize:
- fast loading
- semantic structure
- lightweight assets
- accessibility
- reusable components

Avoid:
- unnecessary dependencies
- bloated JS
- heavy frameworks

---

# Overall Goal

The website should combine:
- emotional reassurance
- calm elegance
- human warmth
- real expertise
- premium softness

# Language Rules

The primary language of the website is French.

All default website content should be generated in:
- French
- natural native-level French
- emotionally intelligent French

Avoid:
- translated English structures
- literal translations
- startup-style English phrasing

English may occasionally be used for:
- technical development
- internal documentation
- code comments
- metadata references

But public-facing content should default to French unless explicitly requested otherwise.
